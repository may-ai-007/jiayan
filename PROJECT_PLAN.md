## 完整用户流程

```
首页输入问题/点击案例
  → /flow/?q=问题
    → 步骤1: 调研规划中（约3s）
    → 步骤2: 数据采集中（约4s）
    → 步骤3: 分析生成中（约3s）
    → 自动跳转 /report/[id]/
      → 完整报告展示（5章 + 图表 + 证据）
      → 跳转 /report/[id]/sources/ 查看原始数据
```
